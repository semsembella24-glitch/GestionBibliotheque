
Gestion de Bibliothèque en Python

 Description
Ce projet est un système de gestion de bibliothèque développé en Python.  
Il permet de gérer des livres, des utilisateurs (adhérents) et les emprunts, en respectant une architecture orientée objet claire et professionnelle. jai utilisé les nomes suivant livre (classe book).utiisateur( classe user.)emprunt( class loan).bibliotheque(class library)

Le projet s’appuie sur des classes distinctes pour représenter les entités principales et inclut un diagramme UML pour illustrer leurs relations.

 Objectifs du projet

- Appliquer les principes de la programmation orientée objet (POO)
- Structurer un projet Python de manière professionnelle
- Gérer les relations entre livres, utilisateurs et emprunts
- Utiliser GitHub pour le versionnement et le partage du projet
